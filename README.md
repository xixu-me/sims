# Student Information Management System

学生信息管理系统

- Student.h：这个头文件定义了一个名为 Student 的类，它是一个学生类，用于存储学生信息。
- Student.cpp：这个源文件实现了 Student 类中的函数。
- Administrator.h：这个头文件定义了一个名为 Administrator 的类，它是一个管理员类，用于管理学生信息。
- Administrator.cpp：这个源文件实现了 Administrator 类中的函数。
- main.cpp：这个源文件包含程序的主函数，它创建了一个 Administrator 对象并调用了它的函数来管理学生信息。

在这个程序中，Administrator 类用于管理学生信息，Student 类用于存储学生信息。程序的主函数创建了一个 Administrator 对象并调用了它的函数来管理学生信息。
